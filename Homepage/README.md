# Emil Berglund Portfolio

Personlig portfolio-nettside bygget med Quasar og Vue 3.
Nettsiden viser prosjekter, ferdigheter, erfaring og kontaktinformasjon i et responsivt oppsett.

## Live Portfolio

[https://emilb.no](https://emilb.no)

## Naviger til mappen

```bash
cd homepage
```

## Install dependencies

```bash
bun install
```

## Start development mode

```bash
bun run dev
```

## Lint files

```bash
bun run lint
```

## Format files

```bash
bun run format
```

## Build for production

```bash
bun run build
```

## Customize configuration

See [Configuring quasar.config.js](https://v2.quasar.dev/quasar-cli-vite/quasar-config-js).

## Siste oppdateringer

- Footer modernisert med tydeligere visuell struktur og GitHub-ikon/lenke.
- Footer bruker nå samme maksbredden som resten av siden for konsekvent layout.
- Forbedret mobiltilpasning i footer (bedre spacing og sentrert CTA).

## Portfolio-nettside

Dette er kildekoden for min personlige portfolio-nettside, bygget med
[Quasar Framework](https://quasar.dev/) og Vue.js. Nettsiden presenterer
prosjekter, ferdigheter og erfaringer pa en moderne og responsiv mate.

## Teknologier brukt

- [Vue.js](https://vuejs.org/)
- [Quasar Framework](https://quasar.dev/)
- TypeScript
- PostCSS
- Bun (pakkebehandler)
- ESLint og Prettier

## Prosjektstruktur

```text
homepage/
|-- public/           # Statisk innhold (ikoner, favicon, osv.)
|-- src/              # Kildekode (komponenter, sider, layouts, osv.)
|-- .quasar/          # Quasar-genererte filer
|-- .vscode/          # VS Code-prosjektinnstillinger
|-- package.json      # Avhengigheter og scripts
|-- quasar.config.js  # Quasar-konfigurasjon
|-- tsconfig.json     # TypeScript-konfigurasjon
`-- README.md         # Denne filen
```

## Funksjoner

- Responsivt design for alle enheter
- Moderne og brukervennlig UI
- Klart skille mellom innhold og presentasjon
- Enkel a tilpasse og utvide
