# 🌐 Portfolio Nettside

Dette er kildekoden for min personlige portfolio nettside, bygget med moderne webteknologier.

**Se nettsiden live:** [emilb.pages.dev](https://emilb.pages.dev)

---

## 🚀 Teknologier brukt

- [Vue.js](https://vuejs.org/)
- [Quasar Framework](https://quasar.dev/)
- TypeScript og JavaScript
- SCSS (Sass)
- Bun (pakkehåndterer)
- ESLint & Prettier

---

## ✨ Funksjoner

- Responsivt design for alle enheter
- Moderne og brukervennlig UI
- Klart skille mellom innhold og presentasjon
- Enkel å tilpasse og utvide

---

## 🛠️ Kom i gang

1. **Installer avhengigheter:**

   ```sh
   bun install
   ```

2. **Start utviklingsserver:**

   ```sh
   quasar run dev
   ```
